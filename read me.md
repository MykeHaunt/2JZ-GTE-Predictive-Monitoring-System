## 2JZ-GTE Turbo Monitoring System

Real-time monitoring and predictive maintenance system for 2JZ-GTE sequentially turbocharged engines.

## Features

Real-time sensor monitoring
Predictive failure analysis
Interactive dashboards
Synthetic data generation
LSTM-based anomaly detection
Prophet forecasting


## Installation

git clone https://github.com/MykeHaunt/2JZ-GTE-Turbo-Monitor.git
cd 2JZ-GTE-Turbo-Monitor
chmod +x setup_and_run.sh
./setup_and_run.sh
