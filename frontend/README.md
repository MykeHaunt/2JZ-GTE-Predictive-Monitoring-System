### Deployment and Usage 🧑‍💻

•	Local Testing:
Open index.html in your web browser. You should see the dashboard header and a chart displaying sample engine temperature data.

•	Integration with Backend:
If your predictive monitoring system provides an API endpoint, update the commented-out fetch code in app.js with the correct URL. This will dynamically update the chart with real-time data.

•	Hosting Options:
You can host this front end application using GitHub Pages, a static site hosting service, or integrate it into your main repository. To use GitHub Pages, push the frontend directory to a branch (e.g., gh-pages) and configure your repository’s settings accordingly.
